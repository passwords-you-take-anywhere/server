# PYTA Backend

PYTA is a FastAPI-based backend focused on secure authentication and encrypted credential storage.

## Core Features

- Email + password authentication
- Secure password hashing
- Session-based authentication
- PostgreSQL database
- SQLModel ORM
- Cookie-based sessions

## Tech Stack

- FastAPI
- SQLModel + SQLAlchemy
- PostgreSQL
- Passlib
- Pydantic Settings
